# xjtu-idle-board

西安交通大学闲置公告板Web项目

## 技术栈

- HTML CSS JavaScript
- Bootstrap
- jQuery
- AJAX
- JSON Web Token
- Nodejs, Express, EJS

node_modules不上传, 可使用`npm install`补全模块

## 运行调试

- Linux/MacOS: `DEBUG=myapp:* npm start`
- Windows: `set DEBUG=myapp:* & npm start`
